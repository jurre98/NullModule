# NullModule (16HP)

We are making a collaborative 16HP blank panel. The idea is to combine everyone's artwork into a single physical Eurorack blank panel. 

Want to add your design? Here is how to do it:

**1. Get the template**
Download `16HP_blankPanel(layout).svg` from the `template/` folder in this repo.

**2. Add your art**
Open the file in Inkscape or Illustrator. 
* Please only draw inside the `Artwork` layer.
* Leave the `Base & Mounting Holes` and `Safe Area` layers locked so the hardware dimensions stay exact. 
* Keep your design within the safe margins so it doesn't get cut off. 

**3. Submit**
Save your work as an `.svg` and name the file after your Discord username (for example: `bakjuh.svg`). 
You can just drop the file in our Discord channel, and we will upload it to the `submissions/` folder here to keep things organized. 

*(If you are used to GitHub, feel free to just make a Pull Request directly to the submissions folder).*
